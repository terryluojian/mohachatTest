DETAILS

MOHA Chat is a free and open source chat solution. Comes with intuitive
web client. Less resource consuming on server side and fast on client side.
Can run on most shared hosts. Only requires Apache with PHP and MySQL on
server side.


PLUG INS

Plug ins are available for MOHA Chat to improve the chat experience of the
end users. All plug ins are available in the mohachat-bundle package. For
users who downloaded mohachat package, can choose the plug ins they are
going to install.

Hope you will find plug ins useful and interesting.
